# P4-COR016: AISuggestion (ai_suggestion_engine)

## Description

The `AISuggestion` class is a lightweight python `dataclass` representing a single, actionable timeline or parameter mutation proposed by the background AI engine. It acts as the immutable data payload sent from the analysis thread to the UI/Engine thread.

## Public Interface Requirements

```python
from dataclasses import dataclass, field
from enum import Enum
from typing import Optional, Dict, Any

class SuggestionType(Enum):
    ADD_KEYFRAME = "add_keyframe"
    MODIFY_KEYFRAME = "modify_keyframe"
    CREATE_ENTANGLEMENT = "create_entanglement"
    APPLY_QUANTUM_EFFECT = "apply_quantum_effect"

class SuggestionPriority(Enum):
    CRITICAL = "critical"
    HIGH = "high"
    MEDIUM = "medium"
    LOW = "low"

@dataclass
class AISuggestion:
    """
    Data payload representing an actionable AI-generated proposal.
    """
    type: SuggestionType
    priority: SuggestionPriority
    path: str
    time: Optional[float] = None
    value: Optional[float] = None
    reason: str = ""
    confidence: float = 0.5
    
    def to_dict(self) -> Dict[str, Any]:
        """Serializes the suggestion for WebSocket transmission."""
        pass
```

## Implementation Notes

- **Immutability:** Once generated by the `AISuggestionEngine`, this payload should be treated as read-only.
- **Serialization:** The `to_dict()` method must exclusively use native Python types (strings, floats) so it can be seamlessly JSON-stringified and broadcasted to the VJLive3 React frontend.
