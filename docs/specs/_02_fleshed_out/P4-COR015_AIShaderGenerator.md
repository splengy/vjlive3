## Description

The `AIShaderGenerator` is a utility class connecting the VJLive environment to an external LLM (OpenAI GPT-4) to dynamically write, mutate, or compile GLSL 330 core fragment shaders at runtime based on natural language descriptions.

## Implementation Notes

- **Sandboxing:** Code generated by the LLM is untrusted. The resulting string must be passed through the `sandbox.py` or specifically the `ShaderCompiler` to ensure it only interacts with authorized uniforms (`time`, `tex`, `resolution`) and doesn't contain infinite loops (though GLSL limits this inherently).
- **Asynchronous Execution:** This node makes external network requests and must definitively run asynchronously to avoid rendering stutter on the main OpenGL thread.
