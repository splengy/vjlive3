## What This Module Does

The `PrismRealmDatamoshEffect` simulates viewing video through a complex crystalline medium — like a faceted diamond or prism — that refracts, disperses, and distorts light. It combines physical optics (Snell's law, chromatic dispersion) with geometric faceting to create a shimmering, rainbow-laden distortion.

**Core Concept:**
- **Refraction:** Light bends when passing through the crystal, displacing UV coordinates
- **Dispersion:** Different wavelengths (RGB channels) refract by different amounts, creating spectral separation (rainbows)
- **Faceting:** The crystal surface is modeled as a tessellation of flat facets; each facet has its own orientation and displacement
- **Caustics:** Focused light patterns from internal reflections add brightness variations
- **Dual input:** Can blend between two video sources (A and B) with the refraction applied

**Key Characteristics:**
- Dual video input (`tex0` and optional `tex1`)
- Depth-aware: uses depth buffer to modulate refraction strength based on scene depth
- Spectral dispersion: RGB channels split apart
- Facet shimmer: animated noise perturbs facet orientations over time
- High GPU tier: requires multiple texture samples and complex arithmetic per fragment
- Real-time performance: optimized for 60fps at 1080p on modern GPUs

---

## What This Module Does NOT Do

- **NOT** a full volumetric ray-tracing: operates in 2D screen space with approximate refraction
- **NOT** a physically accurate dispersion model: uses simplified wavelength separation (RGB only)
- **NOT** a 3D crystal mesh: facets are procedurally generated, not from a 3D model
- **NOT** a caustic photon mapper: caustics are approximated with procedural patterns
- **NOT** a single-source effect: can use dual inputs but works with single input as well
- **NOT** a depth-from-stereo: requires external depth buffer; does not compute depth

## Detailed Behavior and Parameter Interactions

### Core Algorithm

The effect implements **screen-space crystalline refraction** with the following stages:

1. **Facet Field Generation**
   - The screen is divided into a grid of facets (polygons, typically triangles or quadrilaterals)
   - Each facet has a normal vector `N_f` that determines its orientation
   - Facet normals are perturbed by `facet_shimmer` using time-varying noise
   - `facet_density` controls the number of facets (higher = smaller facets)

2. **Refraction Vector Computation**
   - For each UV coordinate, determine which facet it belongs to
   - Compute the incident light direction (approximated as `(0,0,1)` for orthographic view)
   - Apply Snell's law: `η = sin(θ_i) / sin(θ_t)` where η is refractive index
   - Refract the view direction through the facet: `T = refract(I, N_f, refraction_index)`
   - `refraction` parameter controls the refractive index (e.g., 1.0 = no refraction, 2.0 = strong)

3. **Spectral Dispersion**
   - The refractive index depends on wavelength: `η(λ) = η_base + dispersion * (λ - λ0)`
   - In practice, we simulate this by applying different refraction amounts to R, G, B channels
   - `dispersion` parameter controls the strength of chromatic separation
   - Red, green, blue are offset by different amounts along the refraction vector

4. **Depth Modulation**
   - If a depth texture is provided, refraction strength varies with depth
   - Foreground objects (low depth) may have different refraction than background
   - `depth_refraction` parameter scales refraction based on depth value

5. **Caustics Pattern**
   - `caustics_power` controls the intensity of caustic-like bright patterns
   - Caustics are generated by focusing/defocusing of refracted rays, approximated with a Voronoi or noise pattern
   - They add high-frequency brightness variation to simulate light concentration

6. **Geometry Mix**
   - `geometry_mix` blends between pure refraction and a geometric pattern (e.g., sacred geometry overlay)
   - When > 0, a secondary pattern (rotating mandalas or tessellations) is mixed in
   - This adds a "sacred geometry" aesthetic as mentioned in legacy description

7. **Dual Input Blending**
   - If `hasDual = true`, `tex0` is the primary source and `tex1` is secondary
   - The refracted UVs are used to sample from the appropriate texture
   - `u_mix` controls blend between the two sources before refraction

8. **Final Composition**
   - Refracted colors from R, G, B channels are combined
   - Caustics are added (or multiplied)
   - Result is mixed with original (unrefracted) based on `u_mix` (effect strength)

### Parameter Interactions

| Parameter | Type | Range | Default | Description |
|-----------|------|-------|---------|-------------|
| `refraction` | float | [1.0, 5.0] | 2.0 | Refractive index (1.0 = no bend, 2.0 = strong) |
| `dispersion` | float | [0.0, 2.0] | 0.5 | Chromatic separation strength (0 = no rainbow) |
| `facet_density` | float | [1.0, 64.0] | 8.0 | Number of facets per screen (higher = finer) |
| `facet_shimmer` | float | [0.0, 1.0] | 0.3 | Amplitude of facet normal animation |
| `caustics_power` | float | [0.0, 3.0] | 1.0 | Brightness of caustic patterns |
| `geometry_mix` | float | [0.0, 1.0] | 0.0 | Blend with sacred geometry overlay |
| `depth_refraction` | float | [0.0, 2.0] | 1.0 | Depth-based refraction modulation |
| `time` | float | — | 0.0 | Animation time (for shimmer) |

**Parameter Dependencies:**
- `refraction` must be ≥ 1.0 (air to denser medium); values < 1.0 are clamped
- `dispersion` is multiplied by a wavelength-dependent offset; typical values 0.1-1.0
- `facet_density` controls grid resolution; high values (>32) may impact performance
- `facet_shimmer` requires `time` uniform to animate; if `time` static, shimmer is static
- `depth_refraction` only active if depth texture provided; otherwise ignored

### Dual Input Handling

- **`hasDual` flag:** Determines whether `tex1` is used
  - `hasDual = true`: Sample from `tex0` or `tex1` based on some criterion (e.g., depth, noise) or mix them before refraction
  - `hasDual = false`: Use only `tex0`
- The legacy snippet shows: `original = hasDual ? texture(tex1, uv) : texture(tex0, uv);` suggesting that when dual, the "original" for mixing comes from `tex1`, while refraction is applied to `tex0`. This is a design decision: Video A (tex0) is the "realm" being viewed, Video B (tex1) is the "outside world" that appears in non-refracted areas.

### Class Definition

```python
from vjlive3.render.effect import Effect
from vjlive3.render.program import RenderPipeline
import numpy as np

class PrismRealmDatamoshEffect(Effect):
    """
    Prism Realm Datamosh — Crystalline Light Refraction.
    
    Simulates looking through a faceted diamond or crystal.
    Heavily uses spectral dispersion (rainbows) and geometric refraction.
    
    Parameters:
        refraction (float): Refractive index [1.0, 5.0]
        dispersion (float): Chromatic separation strength [0.0, 2.0]
        facet_density (float): Facet count per screen [1.0, 64.0]
        facet_shimmer (float): Facet normal animation amplitude [0.0, 1.0]
        caustics_power (float): Caustic brightness multiplier [0.0, 3.0]
        geometry_mix (float): Sacred geometry overlay blend [0.0, 1.0]
        depth_refraction (float): Depth modulation strength [0.0, 2.0]
        time (float): Animation time in seconds (updated externally)
    """
    
    METADATA = {
        "name": "PrismRealmDatamoshEffect",
        "spec": "P3-EXT131",
        "version": "1.0.0",
        "tier": "Pro-Tier Native",
        "category": "datamosh",
        "description": "Crystalline refraction with spectral dispersion and faceted geometry",
        "gpu_tier": "HIGH",
        "inputs": ["tex0", "tex1 (optional)", "depth_tex (optional)"],
        "outputs": ["fragColor"]
    }
    
    def __init__(self, name: str = "prism_realm_datamosh") -> None:
        fragment_source = self._get_wgsl_shader()
        super().__init__(name, fragment_source)
        
        self.parameters = {
            "refraction": 2.0,
            "dispersion": 0.5,
            "facet_density": 8.0,
            "facet_shimmer": 0.3,
            "caustics_power": 1.0,
            "geometry_mix": 0.0,
            "depth_refraction": 1.0,
            "time": 0.0,
        }
        
        # State for facet normals (could be stored in a texture for efficiency)
        self._facet_state = None
    
    def _get_wgsl_shader(self) -> str:
        """Return WGSL shader source."""
        return """
        @group(0) @binding(0) var tex0: texture_2d<f32>;
        @group(0) @binding(1) var tex1: texture_2d<f32>;
        @group(0) @binding(2) var s0: sampler;
        @group(0) @binding(3) var depth_tex: texture_2d<f32>;  // optional
        @group(0) @binding(4) var depth_sampler: sampler;
        
        struct Uniforms {
            refraction: f32,
            dispersion: f32,
            facet_density: f32,
            facet_shimmer: f32,
            caustics_power: f32,
            geometry_mix: f32,
            depth_refraction: f32,
            time: f32,
            mix: f32,
            enabled: f32,
            has_dual: f32,
        };
        
        @binding(5) @group(0) var<uniform> uniforms: Uniforms;
        
        // Hash function for facet ID
        fn hash2(p: vec2<f32>) -> vec2<f32> {
            var p2 = fract(p * vec2<f32>(123.45, 678.91));
            p2 += dot(p2, p2 + 45.67);
            return fract((p2.xx + p2.yy) * p2.xy);
        }
        
        // Value noise for shimmer
        fn noise2(p: vec2<f32>) -> f32 {
            let i = floor(p);
            let f = fract(p);
            let a = hash2(i).x;
            let b = hash2(i + vec2<f32>(1.0, 0.0)).x;
            let c = hash2(i + vec2<f32>(0.0, 1.0)).x;
            let d = hash2(i + vec2<f32>(1.0, 1.0)).x;
            let u = f * f * (3.0 - 2.0 * f);
            return mix(mix(a, b, u.x), mix(c, d, u.x), u.y);
        }
        
        // Refract vector for given wavelength offset
        fn refract_offset(I: vec3<f32>, N: vec3<f32>, eta: f32, dispersion_offset: f32) -> vec3<f32> {
            // Effective eta = base refraction + dispersion * offset
            let effective_eta = eta + dispersion_offset;
            // Snell's law: sin(θ_i) / sin(θ_t) = eta
            // For simplicity, we approximate 2D refraction in screen space
            let cos_i = dot(I, N);
            let sin_i = sqrt(1.0 - cos_i * cos_i);
            let sin_t = sin_i / effective_eta;
            if (sin_t > 1.0) {
                return I;  // Total internal reflection - could reflect instead
            }
            let cos_t = sqrt(1.0 - sin_t * sin_t);
            return effective_eta * I + (effective_eta * cos_i - cos_t) * N;
        }
        
        @fragment
        fn main(@location(0) uv: vec2<f32>) -> @location(0) vec4<f32> {
            // Facet grid: each facet is a cell in a subdivided grid
            let grid_size = uniforms.facet_density;
            let cell = floor(uv * grid_size);
            let cell_uv = fract(uv * grid_size);
            
            // Generate pseudo-random facet normal from cell ID
            let seed = cell + uniforms.time * 0.1;  // animate with time
            var n = hash2(seed) * 2.0 - 1.0;  // [-1,1]^2
            let normal_2d = normalize(n);
            // Convert to 3D normal (assuming view direction is (0,0,1))
            var N = vec3<f32>(normal_2d, sqrt(1.0 - dot(normal_2d, normal_2d)));
            
            // Facet shimmer: perturb normal with higher-frequency noise
            if (uniforms.facet_shimmer > 0.0) {
                let shimmer = noise2(uv * 20.0 + uniforms.time) * uniforms.facet_shimmer;
                N = normalize(N + vec3<f32>(shimmer, shimmer, 0.0));
            }
            
            // Incident vector (approximate orthographic view)
            let I = vec3<f32>(0.0, 0.0, 1.0);
            
            // Compute refraction offset for each channel with dispersion
            // Red: longer wavelength -> less refraction (smaller eta)
            // Blue: shorter wavelength -> more refraction (larger eta)
            let eta_base = uniforms.refraction;  // e.g., 1.5 for glass
            let dispersion_strength = uniforms.dispersion;
            // Offsets: red = -0.5, green = 0, blue = +0.5 (scaled by dispersion)
            let offset_r = -0.5 * dispersion_strength;
            let offset_g = 0.0;
            let offset_b = +0.5 * dispersion_strength;
            
            var T_r = refract_offset(I, N, eta_base, offset_r);
            var T_g = refract_offset(I, N, eta_base, offset_g);
            var T_b = refract_offset(I, N, eta_base, offset_b);
            
            // Convert 3D refraction vectors to 2D UV offsets (perspective projection)
            // Simple: use XY components scaled by a factor
            let scale = 0.05;  // empirical scale to keep offsets in [0,1] range
            let uv_offset_r = T_r.xy * scale;
            let uv_offset_g = T_g.xy * scale;
            let uv_offset_b = T_b.xy * scale;
            
            // Depth modulation: if depth provided, adjust offset magnitude
            var depth_mod = 1.0;
            if (uniforms.depth_refraction > 0.0) {
                let depth = textureSample(depth_tex, depth_sampler, uv).r;
                // Example: shallower depth (lower value) = more refraction
                depth_mod = 1.0 - depth * uniforms.depth_refraction;
            }
            
            // Determine which texture to sample
            let use_dual = uniforms.has_dual > 0.5;
            // For simplicity, we'll sample tex0 for refracted, tex1 for original if dual
            // But we need to decide per-pixel which source to use; could be based on a mask
            // Here we assume: refracted from tex0, original from tex1 if dual, else tex0
            
            // Sample with per-channel offsets
            var color_r = textureSample(tex0, s0, uv + uv_offset_r * depth_mod);
            var color_g = textureSample(tex0, s0, uv + uv_offset_g * depth_mod);
            var color_b = textureSample(tex0, s0, uv + uv_offset_b * depth_mod);
            var refracted = vec4<f32>(color_r.r, color_g.g, color_b.b, 1.0);
            
            // Caustics: add a bright pattern based on facet orientation
            var caustics = 0.0;
            if (uniforms.caustics_power > 0.0) {
                // Simple caustic: intensity based on normal's z component (facet facing viewer)
                let facing = max(0.0, N.z);
                caustics = pow(facing, 8.0) * uniforms.caustics_power;
            }
            refracted.rgb += caustics;
            
            // Geometry mix: overlay a sacred geometry pattern (e.g., rotating mandala)
            var geometry = vec4<f32>(0.0);
            if (uniforms.geometry_mix > 0.0) {
                // Simple pattern: concentric circles with rotation
                let center = vec2<f32>(0.5, 0.5);
                let d = length(uv - center);
                let angle = atan2(uv.y - 0.5, uv.x - 0.5) + uniforms.time * 0.2;
                let pattern = sin(d * 30.0 + sin(angle * 6.0) * 0.5) * 0.5 + 0.5;
                geometry = vec4<f32>(pattern, pattern, pattern, 1.0);
            }
            refracted = mix(refracted, geometry, uniforms.geometry_mix);
            
            // Original (unrefracted) source
            let original = use_dual ? textureSample(tex1, s0, uv) : textureSample(tex0, s0, uv);
            
            // Final mix
            let output = mix(original, refracted, uniforms.mix);
            
            return output;
        }
        """
    
    def apply_uniforms(self, pipeline: RenderPipeline) -> None:
        """
        Upload current parameter values to the GPU pipeline.
        
        Args:
            pipeline: RenderPipeline instance to update
        """
        pipeline.set_uniform("refraction", self.parameters["refraction"])
        pipeline.set_uniform("dispersion", self.parameters["dispersion"])
        pipeline.set_uniform("facet_density", self.parameters["facet_density"])
        pipeline.set_uniform("facet_shimmer", self.parameters["facet_shimmer"])
        pipeline.set_uniform("caustics_power", self.parameters["caustics_power"])
        pipeline.set_uniform("geometry_mix", self.parameters["geometry_mix"])
        pipeline.set_uniform("depth_refraction", self.parameters["depth_refraction"])
        pipeline.set_uniform("time", self.parameters["time"])
        # mix, enabled, has_dual are managed by base class or external
    
    def set_dual_input(self, has_dual: bool) -> None:
        """Enable or disable dual video input."""
        self.parameters["has_dual"] = 1.0 if has_dual else 0.0
```

### Data Flow

1. Determine facet for current UV via grid subdivision
2. Generate pseudo-random facet normal from cell ID + time
3. Apply shimmer perturbation to normal
4. Compute refraction vector for R, G, B using Snell's law with dispersion offsets
5. Convert 3D refraction to 2D UV offsets
6. Apply depth modulation if depth texture provided
7. Sample `tex0` at offset UVs (per channel)
8. Add caustics based on facet orientation
9. Optionally mix with sacred geometry pattern
10. Blend with original (`tex0` or `tex1` depending on `has_dual`) using `mix`
11. Output final color

## Edge Cases and Error Handling

### Total Internal Reflection (TIR)

- **Scenario:** When `sin(θ_i) / η > 1`, Snell's law yields no solution; light cannot exit the crystal
- **Behavior:** Current shader returns incident vector (no refraction) as fallback; could instead compute reflection
- **Mitigation:** Clamp `refraction` parameter to avoid extreme angles; or implement reflection path for TIR

### Extreme Refraction Values

- **`refraction` too high (>5):** Can cause large UV offsets, leading to sampling outside texture bounds
- **Mitigation:** Clamp `refraction` to [1.0, 5.0] in Python; shader should also clamp offset magnitude

### Facet Density Performance

- **`facet_density` very high (>64):** Increases noise function calls and may cause cache thrashing
- **Mitigation:** Clamp to reasonable range [1, 64]; default 8 is safe

### Missing Depth Texture

- **Scenario:** `depth_refraction > 0` but no depth texture bound
- **Behavior:** Shader treats missing depth as 0 or 1? Should default to 1.0 (no modulation) to avoid unexpected behavior
- **Mitigation:** Document that depth is optional; if `depth_refraction` nonzero, provide depth texture

### Dual Input Missing

- **Scenario:** `hasDual = true` but `tex1` not bound
- **Behavior:** Shader may sample invalid texture; engine should ensure valid binding
- **Mitigation:** Python-side check in `apply_uniforms()` or engine binding logic

### Caustics Artifacts

- **`caustics_power` too high:** Can cause blown-out highlights, saturation clipping
- **Mitigation:** Clamp to [0, 3.0]; recommend ≤ 1.5 for natural look

## Mathematical Formulations

### Facet Grid

```
grid_size = facet_density
cell = floor(uv * grid_size)
cell_uv = fract(uv * grid_size)
```

Each cell defines a facet with a pseudo-random normal.

### Facet Normal Generation

```
seed = cell + time * 0.1
n = hash2(seed) * 2.0 - 1  // [-1,1]^2
N_2d = normalize(n)
N = (N_2d.x, N_2d.y, sqrt(1 - |n|^2))  // 3D unit normal
```

### Refraction (Snell's Law)

Given incident direction `I` (pointing into surface), normal `N`, and refractive index `η`:

```
cos_i = dot(I, N)
sin_i = sqrt(1 - cos_i^2)
sin_t = sin_i / η
if sin_t > 1:
    // Total internal reflection - return I or compute reflection
else:
    cos_t = sqrt(1 - sin_t^2)
    T = η * I + (η * cos_i - cos_t) * N
    T = normalize(T)
```

### Dispersion

```
eta_red = eta_base - 0.5 * dispersion
eta_green = eta_base
eta_blue = eta_base + 0.5 * dispersion

T_red = refract(I, N, eta_red)
T_green = refract(I, N, eta_green)
T_blue = refract(I, N, eta_blue)
```

### UV Offset

```
offset_2d = T.xy * scale
scale = 0.05  // empirical, could be parameter
```

### Depth Modulation

```
depth = depth_tex.r  // [0,1]
depth_mod = 1.0 - depth * depth_refraction
final_offset = offset_2d * depth_mod
```

### Caustics Approximation

```
facing = N.z  // z component of normal (0 to 1)
caustics = facing^8 * caustics_power
```

### Sacred Geometry Pattern (simplified)

```
center = (0.5, 0.5)
d = length(uv - center)
angle = atan2(uv.y-0.5, uv.x-0.5) + time*0.2
pattern = sin(d * 30.0 + sin(angle*6.0)*0.5) * 0.5 + 0.5
geometry = vec4(pattern, pattern, pattern, 1)
```

### Final Mix

```
output = mix(original, refracted_plus_geometry, mix_factor)
```

## Performance Characteristics

### Computational Complexity

- **Per-fragment operations:**
  - 3-4 texture samples (tex0, tex1 optional, depth optional)
  - 2 `hash2()` calls (for facet normal and shimmer)
  - 1 `noise2()` call if `facet_shimmer > 0`
  - 3 full refraction calculations (for R, G, B) each involving `sqrt`, `dot`, `mix`
  - 1 `pow()` for caustics
  - Geometry pattern: `sin`, `atan2`, `length` (if `geometry_mix > 0`)
  - Several `mix()` operations
- **Total:** ~100-150 FLOPs in worst case (all features enabled)
- **Memory bandwidth:** 3-5 texture fetches per fragment

### GPU Utilization

- **Fragment-bound:** Heavy shader; performance scales with resolution
- **Texture bandwidth:** Moderate (3-5 samples)
- **ALU intensity:** High due to trigonometry and square roots
- **No vertex processing:** Full-screen quad

### Optimization Opportunities

- **Precompute facet normals in a texture:** Instead of generating per-fragment, store in a tiled texture to reduce hash computations
- **Approximate refraction:** Use a simple offset based on normal's XY components instead of full Snell's law
- **Reduce dispersion to 2 samples:** Sample at two wavelengths and interpolate, instead of three separate samples
- **Geometry pattern simplification:** Use a simpler pattern (e.g., radial gradient) to avoid `atan2`

## Technical Notes for Implementation

### Refraction Approximation

The shader uses a simplified 2D refraction model. A more physically accurate approach would:
- Use actual 3D ray direction from camera through pixel
- Intersect with a defined crystal geometry (e.g., tetrahedron or dodecahedron)
- Compute exact refraction at intersection point
- That is too expensive for real-time; our screen-space approximation is sufficient for artistic effect.

### Facet Normal Storage

Generating a random normal per fragment via hash is expensive. Optimization:
- Render facet normals to a small texture (e.g., 256×256) at initialization
- In shader, sample normal from that texture using `cell` as coordinates
- This trades ALU for texture fetch; may be faster on some GPUs.

### Dispersion Wavelengths

We use simple offsets: red = -0.5, green = 0, blue = +0.5 scaled by `dispersion`. This is an approximation of Cauchy's equation. A more accurate model would use actual wavelength-dependent refractive indices (e.g., 1.51 for red, 1.52 for green, 1.53 for blue for glass).

### Caustics Alternatives

The current caustics use `N.z` power. Better approaches:
- Voronoi pattern on facet positions
- Refractive caustics via ray tracing (too expensive)
- Precomputed caustic texture tiled over screen
