## What This Module Does

`AgentState` is a strictly typed data structure (`TypedDict`) that serves as the singular memory payload passed between nodes in the VJLive3 LangGraph orchestrated AI loop. Instead of the orchestrator maintaining complex internal state variables, the entire "consciousness" of the agent for a single evaluation cycle lives in this dictionary. As the workflow progresses through Perceive â†’ Recall â†’ Plan â†’ Execute â†’ Verify, each node reads from and writes to this dictionary, which is then serialized and persisted via LangGraph's checkpointers.

---

## What It Does NOT Do

- Does NOT contain any methods or logic. It is purely a data definition.
- Does NOT replace the long-term `MemoryDB` (which persists across application restarts). `AgentState` represents the *ephemeral* working memory of the current LangGraph thread execution.

---

## Public Interface

```python
from typing import Dict, List, Any, Optional, TypedDict
import numpy as np

class AgentState(TypedDict):
    """The state payload bridging LangGraph nodes during a single perception-action cycle."""
    
    # Core Identity
    agent_id: str
    intent: str
    
    # Perception Data
    context: Dict[str, Any]
    screenshot: Optional[np.ndarray]
    ui_elements: List[Dict[str, Any]]
    
    # Memory and Planning
    precedents: List[Dict[str, Any]]
    planned_action: Optional[Dict[str, Any]]
    
    # Execution Outcomes
    execution_result: Optional[Dict[str, Any]]
    feedback: Optional[float]
    
    # Execution Thread Log
    history: List[Dict[str, Any]]
```

---

## Field Definitions

1. **`agent_id` (str):** The unique UUID or name (e.g., `"v_mimeophon_agent"`) representing the agent thread.
2. **`intent` (str):** The high-level semantic goal for the current cycle (e.g., `"cool_down"`, `"glitch_storm"`), often passed from the `AgentPerformanceBridge`.
3. **`context` (Dict[str, Any]):** Live numerical environmental data collected during perception (e.g., `{"bpm": 130, "audio_energy": 0.8}`).
4. **`screenshot` (Optional[np.ndarray]):** The raw bitmap pixel array grabbed at the start of the cycle, used by the Vision-Language Model.
5. **`ui_elements` (List[Dict]):** Bounding boxes and labels extracted from the screenshot or node graph.
6. **`precedents` (List[Dict]):** Past successful actions retrieved from memory that match the current `context` and `intent`.
7. **`planned_action` (Optional[Dict]):** The JSON action command generated by the VLM (e.g., `{"action_type": "slider", "target": "Feedback", "value": 7.5}`).
8. **`execution_result` (Optional[Dict]):** Status flag confirming whether the timeline engine successfully applied the `planned_action`.
9. **`feedback` (Optional[float]):** The calculated success metric (0.0 - 1.0) applied after execution.
10. **`history` (List[Dict]):** An append-only log of mutations to the dictionary used for LangGraph "Time Travel" rewinds.

---

## Edge Cases and Error Handling

### Serialization Limits
Because this dictionary is persisted to disk by LangGraph's `MemorySaver`, the `screenshot` key (which contains a giant uncompressed `numpy` array) must be explicitly cleared to `None` or converted to a compressed base64 JPEG *before* the cycle terminates and hits the checkpointer. Failure to do so will bloat the SQLite checkpoint database to multiple gigabytes within minutes.
